# Flask API
## Basic Api for Flask Setup using Blogs

### Install requirements
```bash
$ pip install -r requirements.txt
```
##### OR
```bash
$ pip3 install -r requirements.txt
```
### How to Run the App

```bash
$ export FLASK_APP=flaskr
$ export FLASK_ENV=development
$ flask run
```
